# AlgoLatch v0.0.3-preview.1 for macOS and Windows

This prerelease fixes the native AI assistant window lifecycle and makes the
version shown by the application match the GitHub tag and installer filenames.
It provides two packages built from the same source revision:

- `AlgoLatch-0.0.3-preview.1-unsigned-universal.dmg` for Apple Silicon and
  Intel Macs;
- `AlgoLatch-Setup-0.0.3-preview.1-unsigned-x64.exe` for Windows 10/11 x64.

## Important security and license notice

- The macOS application is ad-hoc signed, not Developer ID signed or notarized.
  Gatekeeper may require Control-click → Open or approval in System Settings →
  Privacy & Security.
- The Windows installer is not Authenticode-signed. Windows may display an
  unknown-publisher or SmartScreen warning.
- Download only from this GitHub Release and verify `SHA256SUMS.txt`. Do not
  disable Gatekeeper or SmartScreen globally.
- Current source uses the PolyForm Noncommercial License 1.0.0. Noncommercial
  use is permitted under its terms; commercial use requires a separate written
  license from HAN Chen. Earlier tagged versions retain their included license.

## Fixed since v0.0.2-preview.2

- The AI assistant is now an independent top-level window. It no longer remains
  forced above the main workbench.
- Clicking the AI entry restores a minimized assistant and brings a background
  assistant forward.
- Closing and reopening the assistant creates a clean native window.
- Background project, provider, language and appearance updates no longer
  steal focus from the workbench.
- Normal window bounds are retained instead of persisting a minimized geometry.

## Installation

### macOS

1. Download the Universal DMG and `SHA256SUMS.txt` from this Release.
2. Verify the DMG with `shasum -a 256 --check SHA256SUMS.txt`.
3. Open the DMG and copy **AlgoLatch** to **Applications**.
4. If Gatekeeper blocks the unsigned preview, Control-click the app, choose
   **Open**, and approve it once. A new ad-hoc build may request Documents
   permission again because it has no stable Developer ID identity.

### Windows

1. Download the x64 EXE and `SHA256SUMS.txt` from this Release.
2. Verify the installer with `Get-FileHash` before opening it.
3. Run the one-click per-user installer. No administrator access, external
   compiler, gateway script or `PATH` change is required.

Replacing or uninstalling the application does not remove managed projects in
Documents. API credentials and application state remain in the operating
system user profile.

## Verification performed by the release workflow

- Release-boundary, notices, formatting, type and architecture checks.
- Native AI window unit tests plus Electron regression for close, reopen,
  minimize, restore and background focus behavior.
- macOS Universal build, ad-hoc signature verification, DMG install, launch and
  application probe.
- Windows locked-toolchain verification, real install, launch, native C
  compile/run, uninstall and project-retention checks.
- Independent platform manifests followed by one public SHA-256 manifest over
  the final DMG and EXE.

Rollback is data-preserving: withdraw this prerelease without deleting its tag,
restore the previous application bundle, and keep Documents projects and local
application state unchanged.
