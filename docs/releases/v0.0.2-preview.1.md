# AlgoLatch v0.0.2-preview.1 for Windows

This is the first public Windows 10/11 x64 preview. It is intended for early
testing while the project applies to the SignPath Foundation program.

## Important security notice

- The installer is **not Authenticode-signed**. Windows SmartScreen may show an
  unknown-publisher warning.
- Download only from this GitHub Release and verify `SHA256SUMS.txt` before
  opening the installer.
- This preview does not claim a SignPath Foundation signature.

## Included

- One-click, per-user NSIS installation without administrator access.
- Bundled locked llvm-mingw toolchain; no separate compiler or gateway setup.
- AlgoLatch workbench, free node canvas, C source projection, local execution,
  Trace, analysis, Library, guided lesson and optional user-configured AI.
- Uninstaller. Removing the application preserves projects stored in Documents.

## Verification performed by the release workflow

- Type checking and Windows packaging contract tests.
- Pinned toolchain archive digest and execution-chain verification.
- Installation, launch, project creation and native C compile/run checks.
- Uninstallation and project-retention checks.

Report security issues through [SECURITY.md](../../SECURITY.md), not a public
issue.
